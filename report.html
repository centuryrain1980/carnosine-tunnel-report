🎯 三个核心问题(Word 报告 §1)

  • 50 个 AlphaFold 设计蛋白的底物通道几何特征分布如何(瓶颈半径、通道长度、cluster 数)?
  • 哪些变体在通道几何上最接近 fsl_peptidase 原酶(瓶颈 0.82 Å、长度 8.80 Å)?
  • 从工程改造角度,哪 10 个候选蛋白最值得推荐进入下一轮湿实验验证?

🔬 参照酶 fsl_peptidase 关键指标

  • Avg_BR (瓶颈半径): 0.821 Å — 较窄,允许小分子精确进入
  • Avg_L (通道长度): 8.796 Å — 中等,产物驻留时间适中
  • Cluster 数: 4 — 多个通道,提供替代入口
  • 残基 298 类型: ASP — 带负电,引导产物离开

✅ 7.1 关键结论

  • 23/50 个设计保留了 D298,与 fsl 原酶一致,具备催化功能保守性
  • Top 4 候选(fold_36/46/26/04)综合评分 8.00,并列第一;它们在 BR/L/N 三个维度上达到了最佳平衡
  • TRP298/PHE298(27 个)即使通道几何接近原酶,也不推荐 —— 298 位类型偏离会改变产物引导方向,引发副反应
  • 50 个设计的瓶颈半径普遍宽于 fsl 原酶(40-75%),这是 AlphaFold 预测 vs X-ray 晶体的固有差异

🧪 7.2 推荐湿实验方案

  • 第一轮(优先):对 fold_36 + fold_46 + fold_26 + fold_04 共 4 个 Top 候选同时进行体外表达纯化 + 酶活测定 + HPLC 检测
  • 第二轮(扩展):若第一轮结果与预测吻合,扩展测试 fold_38/16/06/18 共 4 个次优候选
  • 对照实验:测试 1-2 个 TRP298 设计(如 fold_02,通道长度最接近 fsl),验证 298 位类型对选择性的影响

⚠️ 局限性说明(7.3)

本报告基于 AlphaFold 预测结构,实际蛋白折叠可能与预测有偏差。 通道分析是结构筛选,不能替代湿实验验证;Top 10 候选应当通过实际酶活测定确认。 此外,本报告未考虑底物结合亲和力(Kd)、催化速率(kcat)等动力学参数, 建议在湿实验阶段同步测定 Km 和 kcat 以获得完整的酶学表征。

🔬 8.1 通道计算工具

CAVER 3.02 是识别和表征蛋白质内部空腔/通道的工业标准工具, 已被 1000+ 研究组采用。本项目通过 PyMOL 1.8 集成接口调用 CAVER。

引用: Chovancova E, et al. (2012) CAVER 3.0: A Tool for the Analysis of Transport Pathways in Dynamic Protein Structures. PLoS Comput Biol 8: e1002708. DOI: 10.1371/journal.pcbi.1002708

📐 统一 CAVER 参数(所有 50 fold + fsl)

  • 探针半径 probe_radius:0.7 Å(小于水分子,捕获细通道)
  • 起始点保护半径:4.0 Å
  • 壳层半径 / 壳层深度:5.0 Å / 8.0
  • 聚类阈值:3.5 Å
  • 残基接触距离:3.0 Å
  • 起始点:每个蛋白 8 个活性中心残基 Cα 几何中心(独立计算)

⚙️ 8.2 批量执行流程

  1. 50 个 CIF 文件 → 标准库 mmCIF 解析 → 转换为 PDB 格式
  2. 对每个 PDB 独立计算活性中心 Cα 几何中心作为 CAVER 起始点
  3. 生成 CAVER config.txt(统一参数)并启动 CAVER
  4. 50 个 CAVER 任务分 10 批 × 5 个并行执行,Windows Java 后端
  5. 总耗时约 30-60 分钟(并行加速)

🌐 8.3 数据汇总工具

  • 体积计算: V = Σ (4/3)π r³,CAVER 球链结构中 r = B-factor
  • 衬里化学分类:
    • 正电: ARG / LYS / HIS(蓝)
    • 负电: ASP / GLU(红)
    • 极性: SER / THR / ASN / GLN / TYR / CYS(绿)
    • 疏水: ALA / VAL / LEU / ILE / PHE / TRP / MET / GLY(橙)
  • 综合评分: BR(1.0-1.4 Å)+ L(> 4 Å)+ Cluster 数(适中)+ 强制 D298,综合分 0-10

🖥️ 3D 渲染技术

  • 静态缩略图: Python matplotlib 3D(51 张 PNG,本目录 renders/)
  • 交互式查看: 3Dmol.js 2.4.2 加载 PDB + 自定义球链 + 鼠标交互(旋转/缩放/平移)
  • 化学着色: 8 个活性位点 Cα 按残基类型分 4 色,Tunnel 球按 cluster 着色
